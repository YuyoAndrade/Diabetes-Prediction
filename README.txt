Instrucciones para ejecutar el programa
1. Instalar Jupter Notebook con sus extensiones.
2. Abrir el archivo de jupyter "diabetes".
3. Ver las librerias en el primer cuadro e instalarlas.
4. Tener el archivo de la base de datos con el nombre de "diabetes_012_health_indicators_BRFSS2015.csv" en el mismo directorio que el archivo de jupyter
5. Correr el archivo completo de diabetes. No se debe de modificar el archivo de jupyter y dentro viene la explicación de lo que se está haciendo en cada paso.


Base de Datos Obtenida de: https://www.kaggle.com/datasets/alexteboul/diabetes-health-indicators-dataset
